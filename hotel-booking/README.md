# Hotel Room Booking System

A complete hotel room booking website built with HTML, CSS, JavaScript, PHP, and MySQL.

## Features

✅ User registration and login
✅ Room browsing and search/filter
✅ Room details with image slider
✅ Real-time booking system
✅ Admin management dashboard
✅ Responsive mobile design
✅ Payment integration support
✅ Secure authentication
✅ Database management
✅ Modern UI/UX
